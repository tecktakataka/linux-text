P332	8.1	Xのインストールと設定				
						
P332	8.1.1 	GUIを実現する技術				
						
	X Windows System　：　GUIを実現（Linux、Unix）					
						
		GUI　：　Windows、マウスを利用して操作				
		CUI　：　Linux、Unix、コマンドラインでの操作				
						
	別名(X Windows System)　：　X、X11					
						
		バージョン	リリース日	説明	ディストリビューション	備考
		X	1984年6月	Linuxの最初GUI	XFree86	
		X11	1987年9月15日		X.Org	Xfree86の派生
						
		ディストリビューション　：　				
		利用者が容易にインストール・利用できるLINUX（Linuxカーネルとその他ソフトウェア群を1つに纏めたもの）				
						
	X　：　クライアント/サーバ方式					
						
		Xサーバ　：　ハードウェア管理（モニター、ビデオカード、キーボード）				
			Xクライアントにグラフィカルなインターフェースを提供			
		Xクライアント：ユーザアプリケーション（Webブラウザ、オフィスアプリケーション）				
						
		Xサーバ、Xクライントは別々の環境でも利用可能				
						
	X Windows System　：　現在のコンピュータ環境で無理がある、古くからある仕組み					
						
	Wayland　：　対応していないアプリケーションがある、対応していても不十分の場合もある					
		ウィンドウマネージャがディスプレイサーバとしてハードウェアやグラフィックを管理				
						
P333	8.1.2	X.Orgの設定				
						
						
	/etc/X11/xorg.conf		昔、X.Orgの手動設定　			
			現在、手動設定はほぼない			
			LPICでは設定が問われる			
						
	xorg		X Windows System公式のオープンソース（LINUX）			
						
	設定					
	/etc/X11/xorg.conf					
	/etc/X11/xorg.conf.dディレクトリ階層にある.confファイル					
						
	# Xorg -configure		Xorgを自動生成			
			/root配下に、xorg.conf.new　ファイルが作成			
						
	# X -config /root/xorg.conf.new			テスト実行		
						
	# cp /root/xorg.conf.new /etc/X11/xorg.conf				テストファイルを利用しているxorgへコピー

P345	8.2.4	リモートデスクトップ				
						
		リモートデスクトップ　：　ネットワーク経由でPCからPCもしくはサーバへ接続				
						
		仮想デスクトップ環境（VDI）		：個人PC(ローカルPC)をサーバへ接続し、サーバ上にある仮想環境（仮想デスクトップPC）へ接続		
				セキュリティ強化　：　個人PCにデータは格納されず、全てサーバにデータを保管する		
						
		接続方法				
		VNC	Windows、Linux、macOSなど違うOS対応			
		RDP	Windows標準、リモートデスクトップ			
		SPICE	RDP同様、オープンソース、通信暗号化、マルチモニタなど多機能に対応			
		XDMCP	ディスプレイマネージャをネットワーク越しに利用できる
